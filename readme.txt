== Description ==
'Accordion title' lets you use the next container element as a accordion content. Clicking on the 'Accordion title' will show/hide the content.

Add the 'Accordion title' before an 'inner content'  and you can open/close that 'inner content' like the normal accordion.
Inside the editor you will see a red border around the content that the 'Accordion title' will open/close.

== Installation ==
* install [Elementor](https://be.elementor.com/visit/?bta=207645&brand=elementor)
* install the plugin

== How to use it ==

* create a inner content
* Add the 'Accordion title' widget and style it
* Add another 'Inner Content' below the widget (this will be the hide/show content)
* Fill the 'Inner Content' as you like.
